# Rental Car Practice

This is a simple rental car management program made with Python.

## Features

* Create cars
* Rent cars
* Return cars
* Automatically assign an available car
* Show all car information
* Support electric cars

## Classes

### Car

Stores information about one car.

### ElectricCar

Represents an electric car.

### RentalCompany

Manages all rental cars.

## Purpose

I made this project to practice Python classes, inheritance, and object-oriented programming.
